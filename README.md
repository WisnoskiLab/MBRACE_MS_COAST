# MBRACE_MS_COAST

# Transport and Fate of Bacterial Communities in Mississippi Coastal Ecosystems 
- MBRACE (https://mbrace.usm.edu/transport-and-fate-of-bacterial-communities-in-mississippi-coastal-ecosystems)
This repository contains bioinformatics workflows, data organization templates, scripts, and documentation for the **MBRACE  project**. The project focuses on understanding temporal changes in microbial communities and functional potential across estuarine samples using **metagenomics** and **transcriptomics** approaches.

The study includes an initial dataset collected in **November 2025**, followed by a structured **time-series sampling campaign from January to December 2026**. Sampling is planned at every month for **amplicon sequencing** and approximately **three-month intervals**, for **metagenomics** and **transcriptomics** allowing seasonal and temporal changes in microbial community composition, functional genes, and gene expression patterns to be evaluated.

The major goals of this repository are to:

- Organize raw and processed sequencing data
- Maintain reproducible bioinformatics workflows
- Track sample metadata and sequencing outputs
- Support metagenomic and transcriptomic analyses
- Provide a clear structure for collaborative project development
